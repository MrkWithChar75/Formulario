Formulario Limiti - Analisi Matematica

Limiti Notevoli

  • \(\displaystyle \lim_{x \to 0} \frac{\sin x}{x} = 1\)
  • \(\displaystyle \lim_{x \to 0} \frac{\tan x}{x} = 1\)
  • \(\displaystyle \lim_{x \to 0} \frac{1 - \cos x}{x^2} = \frac{1}{2}\)
  • \(\displaystyle \lim_{x \to 0} \frac{\ln(1 + x)}{x} = 1\)
  • \(\displaystyle \lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e\)
  • \(\displaystyle \lim_{x \to 0^+} x^x = 1\)
  • \(\displaystyle \lim_{x \to 0} \frac{e^x - 1}{x} = 1\)
  • \(\displaystyle \lim_{x \to 0} \frac{a^x - 1}{x} = \ln a\)

Ordini di Infinito e Infinitesimo

Infinitesimi per \(x \to 0\)

  • \(\sin x \sim x\)
  • \(\tan x \sim x\)
  • \(1 - \cos x \sim \frac{x^2}{2}\)
  • \(\ln(1 + x) \sim x\)
  • \(e^x - 1 \sim x\)
  • \((1 + x)^n \sim 1 + nx\)

Ordini di infinito per \(x \to \infty\)

\(\ln x \ll x^a \ll a^x \ll x! \ll x^x\)

Regole Operative sui Limiti

  • \(\lim (f(x) + g(x)) = \lim f(x) + \lim g(x)\)
  • \(\lim (f(x) \cdot g(x)) = \lim f(x) \cdot \lim g(x)\)
  • \(\lim \left(\frac{f(x)}{g(x)}\right) = \frac{\lim f(x)}{\lim g(x)}\) se \(\lim g(x) \ne 0\)

Composizione

Se \(f(x) \to L\) e \(g(x) \to f(L)\), allora \(\lim g(f(x)) = g(\lim f(x))\)

Forme Indeterminate

\(\frac{0}{0}, \frac{\infty}{\infty}, \infty - \infty, 0 \cdot \infty, 1^\infty, 0^0, \infty^0\)

Si risolvono con:

  • Razionalizzazione
  • Sviluppi di Taylor
  • Regola di De L’Hôpital:
    Se \(\lim \frac{f(x)}{g(x)} = \frac{0}{0}\) o \(\frac{\infty}{\infty}\), allora:
    \(\lim \frac{f(x)}{g(x)} = \lim \frac{f'(x)}{g'(x)}\)

Confronti Utili

  • Per \(x \to 0\): \(\sin x < x < \tan x\)
  • \(\ln(1 + x) < x < \frac{x}{1 - x}\) per \(x > 0\)

Dominanza: se \(f(x) \sim g(x)\), cioè \(\lim \frac{f(x)}{g(x)} = 1\), si comportano allo stesso modo nei limiti.

Sviluppi di Taylor (1° ordine)

  • \(\sin x = x - \frac{x^3}{6} + \cdots\)
  • \(\cos x = 1 - \frac{x^2}{2} + \cdots\)
  • \(\ln(1 + x) = x - \frac{x^2}{2} + \cdots\)
  • \(e^x = 1 + x + \frac{x^2}{2} + \cdots\)
